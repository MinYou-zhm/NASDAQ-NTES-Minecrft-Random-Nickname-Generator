# 网易我的世界随机昵称生成器

这是一个基于网页的网易我的世界随机昵称生成器

## 功能特点

1. **随机昵称生成**：根据网易我的世界的命名思路，生成独特的昵称
2. **自定义设置**：
   - 字符顺序：可以选择「形容词+名词」或「名词+形容词」
   - 数字位置：可以选择「无数字」、「数字在开头」、「数字在中间」或「数字在结尾」

## 使用方法

1. 直接双击 `minecraft-nickname-generator.html` 文件
2. 在「生成设置」区域选择你喜欢的字符顺序和数字位置
3. 点击「生成」按钮获取随机昵称
4. 生成的昵称会显示在透明圆角输入框中

## 技术实现

- **前端框架**：纯HTML、CSS和JavaScript
- **背景效果**：使用CSS动画实现背景图片的移动效果
- **动画效果**：使用CSS关键帧动画实现昵称的渲染效果
- **随机生成**：使用JavaScript的Math.random()函数生成随机昵称

## 生成规则

1. **形容词**：从预设的25个形容词中随机选择一个
2. **名词**：从预设的35个名词中随机选择一个
3. **数字**：随机生成3位数字
4. **组合方式**：根据用户选择的字符顺序和数字位置进行组合

## 注意事项

- 本生成器仅用于生成类似网易我的世界的随机昵称，不保证生成的昵称一定可用
- 如需添加更多形容词或名词，请修改 `minecraft-nickname-generator.html` 文件中的对应数组
